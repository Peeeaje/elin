
=== Configuration files

User configuration:

* `$XDG_CONFIG_HOME/elin/config.edn`
* `$HOME/.config/elin/config.edn`

Project local configuration:

* `.elin/config.edn`


`:uses` は `:includes` と `:config-map` のシンタックスシュガーで以下と等価です。

[source,edn]
----
{:uses [foo {:bar "baz"}]}

{:includes [foo]
 :config-map {foo {:bar "baz"}}}
----
